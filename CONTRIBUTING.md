# Contributing

This repository is public-facing. Contributions must be public-safe, educational, and reusable.

Before contributing, check:

1. The content does not expose pilot-sensitive data.
2. The content does not include credentials, tokens, private endpoints, raw logs, or raw vulnerability details.
3. The content uses synthetic or sanitized examples.
4. The content explains acronyms at first meaningful use.
5. The content fits the NIS2COMPASS public dissemination purpose.

Preferred contribution types:

- Plain-language documentation.
- Reusable templates.
- Synthetic examples.
- Public-safe schemas.
- Website improvements.
- Typo and clarity fixes.

Do not contribute internal project records, funder communication, payment material, signed annexes, or private pilot evidence.
