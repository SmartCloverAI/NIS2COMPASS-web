# Deliverable Types

| Deliverable type | Meaning | Public example |
| --- | --- | --- |
| Document | A Markdown or PDF explanation. | Public playbook. |
| Template | Empty reusable structure. | Evidence-record CSV template. |
| Schema | Machine-readable data contract. | Vulnerability finding JSON Schema. |
| Dataset | Synthetic or sanitized data sample. | Synthetic monitoring events. |
| Website page | Public-facing project explanation. | Project overview page. |
| Demo asset | Public-safe material used in a demo. | Synthetic evidence pack. |
| Checklist | Step-by-step review list. | Public-release checklist. |
