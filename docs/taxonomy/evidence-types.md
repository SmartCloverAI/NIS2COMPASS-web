# Evidence Types

| Evidence type | Plain meaning | Public-safe example |
| --- | --- | --- |
| Report | Human-readable explanation of findings, work completed, or lessons learned. | Public methodology report. |
| CSV register | Spreadsheet-style list of records. | Empty risk-register template. |
| JSON record | Machine-readable structured evidence. | Synthetic evidence record. |
| JSONL log sample | One JSON record per line. | Synthetic monitoring events. |
| Screenshot | Visual proof that something existed or ran. | Screenshot of a public web page. |
| Schema | Machine-readable rule set for data. | Evidence record JSON Schema. |
| Approval note | Human confirmation that an artifact was reviewed. | Public release checklist result. |
| Hash manifest | File fingerprints used to prove evidence integrity. | Synthetic evidence-pack manifest. |
