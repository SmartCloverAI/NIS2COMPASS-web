# Project Overview

NIS2COMPASS helps organizations improve readiness for the NIS2 Directive and organize evidence that another person can review.

The project addresses a practical problem: security work is often scattered across documents, spreadsheets, screenshots, logs, reports, tools, and meeting notes. Scattered records are hard to review, maintain, and reuse. NIS2COMPASS explores how those records can be connected to controls, owners, decisions, remediation work, and evidence packs while keeping people responsible for final conclusions.

## Relationship With CYberSynchrony

CYberSynchrony is the larger modular cybersecurity project that selected NIS2COMPASS through its Financial Support to Third Parties open call. The [six-module centre](cybersynchrony-alignment.md) explains how the public CYberSynchrony framework covers preparedness, threat intelligence, monitoring, secure exchange, awareness, and governance.

Alignment does not mean that every CYberSynchrony tool, API, sandbox, or private service is publicly available or already integrated with NIS2COMPASS.

## Public Outputs

This repository publishes:

- Plain-language learning material.
- Public-safe playbooks and methods.
- Reusable templates and schemas.
- Synthetic examples containing no real pilot data.
- Public project progress notes.
- Partner publications and authoritative external links.
- Website source and deployment documentation.

It does not publish pilot-sensitive evidence, raw logs, vulnerabilities, credentials, personal data, operational test instructions, private architecture, or internal funder records.
