# Project Overview

NIS2COMPASS helps organizations improve readiness for the NIS2 Directive and prove that improvement with organized evidence.

Public content development for this repository and website is performed by SmartClover SRL.

Copyright belongs to AI STM Learning SRL and SmartClover SRL, as the NIS2COMPASS consortium partners.

NIS2 is the European Union cybersecurity law formally named Directive (EU) 2022/2555. It expects important and essential organizations to manage cyber risk, prepare for incidents, report serious incidents, train people, protect systems, and keep proof that these actions are happening.

NIS2COMPASS focuses on a practical problem: security work is often scattered across documents, spreadsheets, screenshots, logs, reports, and meeting notes. Scattered evidence is hard to review and hard to reuse.

The project turns scattered evidence into structured public-safe methods, templates, and examples.

## Public Outputs

This repository publishes:

- Plain-language learning material.
- Public-safe playbooks.
- Reusable templates.
- Public taxonomy.
- Synthetic examples.
- Website source.

It does not publish pilot-sensitive evidence, raw logs, vulnerability details, personal data, or internal funder records.
