# Evidence Pack Playbook

An evidence pack is a reviewable folder or archive that proves a set of project claims.

## Minimum Contents

- Evidence register.
- File list.
- Hash manifest.
- Review log.
- Summary report.
- Public/private label.

## Good Evidence

Good evidence is specific, dated, owned, reviewable, and linked to a control or deliverable.

## Bad Evidence

Bad evidence is vague, unowned, impossible to open, not dated, or impossible to connect to a claim.
