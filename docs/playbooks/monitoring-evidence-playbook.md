# Monitoring Evidence Playbook

Monitoring evidence shows what was watched, what data was collected, and how suspicious activity would be noticed.

## Public-Safe Outputs

- Monitoring source inventory template.
- Sanitized event example.
- Detection profile summary.
- Monthly monitoring summary structure.

## Do Not Publish

- Raw logs.
- Private hostnames.
- Private IP addresses.
- Usernames.
- Tokens.
- Alert details that expose a real weakness.
