# NIS2 Readiness Starter Guide

This guide helps a team start NIS2 readiness work without assuming deep cybersecurity knowledge.

## Steps

1. Identify important services and systems.
2. Identify who owns each service or system.
3. List existing policies, procedures, tools, and reports.
4. Map those materials to practical NIS2 control areas.
5. Find gaps where no evidence exists.
6. Create a remediation roadmap.
7. Collect evidence as work happens.
8. Review evidence before it is used in reports.

## Minimum Outputs

- Asset or source inventory.
- Control register.
- Risk register.
- Evidence register.
- Remediation roadmap.
- Review and approval notes.
