# Public Dissemination Guide

Public dissemination means sharing useful lessons and artifacts without exposing sensitive information.

## Publish

- Plain-language guides.
- Public-safe templates.
- Synthetic examples.
- Methodology notes.
- Workshop summaries.
- Public screenshots and URLs.

## Do Not Publish

- Pilot-sensitive evidence.
- Raw logs.
- Raw alerts.
- Real vulnerabilities.
- Personal data.
- Credentials.
- Private architecture.

## Review Rule

Every public item should pass the publication checklist before release.
