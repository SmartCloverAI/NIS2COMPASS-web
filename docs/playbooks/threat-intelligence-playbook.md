# Threat-Intelligence Playbook

Threat intelligence is information about threats that helps a team decide what to monitor, test, or prepare for.

## Minimum Steps

1. List relevant threat sources.
2. Explain why each source matters for the sector.
3. Select useful Indicators of Compromise only when they are safe to handle.
4. Map indicators to monitoring or response actions.
5. Use STIX/TAXII examples only with synthetic or public-safe data.

## Public-Safe Outputs

- Threat-feed list template.
- Synthetic STIX example.
- Public response-playbook explanation.
