# CYBERRA

CYBERRA focuses on threat intelligence and risk analysis. It helps teams collect relevant information about threats, add context, compare indicators, and turn that knowledge into better risk and response decisions.

## What This Means For NIS2COMPASS

NIS2COMPASS uses the CYBERRA role to explain how reviewed threat information can support evidence records and human-approved response playbooks. Public examples remain synthetic. No real indicator, organization-specific threat report, private feed, or federation access is published.

## Public Open-Source Reference

<!-- tool-ref:misp -->

**[MISP](https://www.misp-project.org/)** is an open-source platform for creating, enriching, correlating, and sharing structured cyber-threat intelligence. The public CYberSynchrony D3.1 deliverable describes MISP as part of its Enhanced Threat Intelligence System work.

MISP can exchange sensitive information, so availability of the software does not authorize data sharing. A real exchange still requires governance, permitted sources, handling labels, trusted recipients, and a reviewable approval route.
