# CROSS-CORE

CROSS-CORE focuses on secure cooperation and information exchange during cyber incidents. Public CYberSynchrony material also discusses integrity mechanisms that can help detect whether shared records have changed.

## What This Means For NIS2COMPASS

NIS2COMPASS uses the CROSS-CORE role to explain evidence packaging, hashes, controlled exchange, and traceable review. A hash or integrity record can help show that a file changed. It cannot prove that the original file was true, lawfully collected, complete, written by a particular person, legally admissible, or sufficient for NIS2 compliance.

> **Public software position**
>
> NIS2COMPASS does not claim an independently downloadable open-source CROSS-CORE implementation on this page. Readers are directed to the public CYberSynchrony data-exchange and evidence-integrity deliverables for the authoritative project description.
