# Security Policy

This repository is for public dissemination and public-safe examples.

## Reporting A Problem

If you find sensitive information in this repository, report it privately to the maintainers. Do not open a public issue with secrets, pilot details, exploit paths, raw logs, or personal data.

## Public Boundary

The repository must not contain:

- Credentials, tokens, private keys, or passwords.
- Private endpoints or internal architecture.
- Raw security logs.
- Raw vulnerability reports.
- Exploit instructions for real pilot systems.
- Personal data.
- Pilot-sensitive business information.

Synthetic examples are allowed when they are clearly fake and cannot be mistaken for real pilot evidence.
